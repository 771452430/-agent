# 支持问题 Agent 介绍

## 1. 这是什么

支持问题 Agent 是一个把以下能力串起来的自动化助手：

- 飞书多维表格
- 知识库检索
- 大模型总结
- 定时轮巡
- 自动回写结果

它的目标是把支持问题处理中的重复动作自动化，提升效率，减少人工重复劳动。

---

## 2. 它能做什么

支持问题 Agent 目前可以完成下面这些动作：

### 2.1 读取飞书问题台账

- 连接飞书多维表格
- 验证表格地址是否可读
- 读取表格字段
- 预览表格数据
- 验证是否具备写权限

### 2.2 自动筛选待处理问题

系统会自动筛选：

- `回复进度 = 待分析`
- `回复进度 = 失败待重试`

这些数据会进入本轮处理队列。

### 2.3 自动分析问题

对每一条待处理问题，系统会：

1. 先把状态改成 `分析中`
2. 读取 `问题` 列内容
3. 去知识库里检索相关资料
4. 调用大模型基于检索结果做总结
5. 生成 `AI解决方案`

### 2.4 自动补充相关文档链接

- 如果命中的知识文档配置了线上地址
- 系统会把对应的文档链接写入 `相关文档链接`

### 2.5 自动回写处理状态

系统会把处理结果写回飞书表格：

- 成功：`AI分析完成`
- 无命中：`无命中`
- 失败：`失败待重试`

---

## 3. 它的业务价值

### 3.1 提升支持效率

过去支持同学需要自己：

- 查资料
- 整理答案
- 补文档链接
- 维护问题状态

现在这些动作可以由 Agent 自动完成第一版。

### 3.2 提高知识复用率

知识库不只是存文档，而是能真正参与到支持答复中。

### 3.3 形成标准化流程

把支持问题处理过程变成统一流程：

- 登记问题
- AI 初步分析
- 人工确认补充
- 完成处理

### 3.4 全程可追踪

每次运行都有记录，可以看到：

- 本轮读取多少行
- 命中多少待处理数据
- 成功多少
- 失败多少
- 每一条记录的处理结果

- Agent 配置页面

## 5. 整体工作流程

可以按下面的顺序理解：

### 第一步：人工录入问题

支持同学在飞书多维表格里登记问题。

常用字段包括：

- `问题`
- `登记人`
- `备注（文案或截图）`
- `回复进度`
- `AI解决方案`
- `相关文档链接`

### 第二步：系统筛选待处理数据

系统筛选 `回复进度` 为以下值的数据：

- `待分析`
- `失败待重试`

### 第三步：进入处理中

系统先把该条问题状态更新为：

- `分析中`

### 第四步：知识库检索

系统根据 `问题` 列内容去知识库检索相关文档和片段。

### 第五步：LLM 总结

系统把检索到的上下文交给大模型，让模型生成一段适合写回台账的解决方案。

### 第六步：回写结果

系统会回写：

- `AI解决方案`
- `相关文档链接`
- `回复进度`

### 第七步：保留运行记录

本轮处理完成后，会生成一条运行记录。

## 6. 页面怎么演示

给领导演示时，建议按这个顺序操作。

### 6.1 先演示配置

把某条记录的 `回复进度` 设成 `待分析`，再点击 `立即运行`。

可以看到系统自动完成：

- 改状态为 `分析中`

- 补充 `相关文档链接`
- 改状态为 `AI分析完成`

## 7. 当前版本的边界

当前版本主要定位为：

- 支持问题的自动初步分析助手

它适合：

- 给出第一版解决方案
- 帮助支持同学提速
- 统一沉淀处理过程

它不适合：

- 完全替代人工判断
- 脱离知识库做最终结论

另外还有两个实际边界：

### 7.1 依赖知识库质量

如果知识库没有相关内容，就只能返回 `无命中`。

### 7.2 当前链接列是 URL 类型

当前飞书表里的 `相关文档链接` 是 URL 字段，所以当前版本回写的是一条最佳链接。

## 8. 后续可以扩展什么

后续可以继续扩展为部门级 AI 平台，方向包括：

- 支持更多数据源
- 接更多部门 skill
- 自动分类
- 自动指派
- 自动告警
- 自动生成处理建议
- 自动联动其它系统

也就是说，现在这个 Support Agent 是一个起点，未来可以继续扩展成统一的 AI 工作平台。

---

## 9. 汇报时可以直接说的一段话

我们现在做的这个支持问题 Agent，本质上是把飞书台账、知识库检索、大模型总结和自动回写打通成一个闭环。  
过去支持同学需要自己查资料、整理答案、补链接、改状态；现在系统可以先自动完成第一版分析，再由人工确认和补充。  
这样既提升了效率，也提高了知识复用率，同时把整个处理过程留痕下来，后续可以持续沉淀成部门统一的 AI 工作平台。
